# App Escolar

Plataforma web de comunicación escolar diseñada para conectar a profesores y padres de familia, facilitando el seguimiento académico y la comunicación diaria entre la institución y las familias. (Es solo una muetsra de diseño)

## Descripción

App Escolar centraliza la comunicación entre profesores, estudiantes y padres en una sola plataforma. Permite gestionar mensajes, eventos del calendario académico y documentos importantes, con una experiencia adaptada tanto a escritorio como a dispositivos móviles.

## Características principales

- **Dashboard**: Panel principal con resumen de actividad relevante para cada usuario.
- **Chat**: Sistema de mensajería con notificaciones de mensajes no leídos.
- **Calendario**: Visualización de eventos y fechas académicas importantes.
- **Documentos**: Gestión y acceso a documentos compartidos entre profesores y padres.
- **Filtrado por curso**: Cada usuario visualiza únicamente la información correspondiente a su curso o grupo.
- **Diseño responsive**: Navegación adaptada con barra inferior en móviles, similar a Google Classroom.

## Tecnologías usadas

| Tecnología | Uso |
|---|---|
| React | Librería principal para la construcción de la interfaz |
| Vite | Entorno de desarrollo y build |
| MUI (Material UI) | Componentes de interfaz y diseño visual |

## Arquitectura

El proyecto utiliza una capa de datos simulada (mock data) para el desarrollo y pruebas, junto con un `ChatContext` que gestiona el estado global de los mensajes y los indicadores de no leídos en toda la aplicación.
